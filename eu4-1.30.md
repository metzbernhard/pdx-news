# Söldner

+ Feste Söldnerarmeen mit eigenem begrenzten Manpowerpool, eigener General, können nicht geteilt oder gemerged werden, wählen neuen General wenn alter stirbt. 
+ Wenn man z.B. schweizer Garde anheuert, ist sie deswegen **nicht** für andere Nation incht mehr verfügbar!
+ Nutzen weiterhin Land Boni und haben eigene Modifikatoren
+ lokale (generell), immer da, mind. 3
+ fremde (Modifikatoren, haben Heimatprovinz, können in Handelsreichweite angeheuert werden
+ Aufgelöste Armee kann 20 Jahre nicht wieder vom selben Spieler angeheuert werden
+ Min 2, Max 60 Regimenter, abhängig Nation Dev
  ![](http://i.imgur.com/Vj37min.png)
+ Je näher Provinz an Söldnerprovinz desto schneller Rekrutiert
+ Werte v.a. abhängig von der Heimatprovinz (Techgruppe etc)
+ Nicht mehr so teuer, da manpower limitiert, billige Option für kleine mit Nachteilen 
![](https://forum.paradoxplaza.com/forum/index.php?attachments/upload_2020-2-25_8-54-20-png.547914/)
+ werden mit Mil Tech schneller teurer (8% statt 2%) als normale Soldaten
+ ~100 Kompanien, besondere wie Ungarische Black Army oder Schweizer Garde
+ Verfügbare Söldner => SöldnerManpower 
+ Immer 100% Unterhalt

# Revolutionen

+ Kann anfangen wenn absoluter herrschen in großer Nation, pro 30 Entwicklung provinz jährliche Chance, dass sie beginnt (auch außerhalb europas möglich!)
+ Verschiedene Faktoren: Absolutismus, dritter Stand, Revanchismus, geringes Prestige, zu viel Händlerkontrolle, große Schulden
+ Diverse Events, die zur Revolution aufbauen, abhängig von Antworten => höhere Chance, dass man Revolutionszentrum erhält
+ Revolutionszentrum verbreitet Revolutionäre Ideen ähnlich Reformationszentren (deutlich schneller!), hat eine Provinz Revolutionäre Ideen erhält es die Hälfte des Absolutisums als +Autonomie, Untertanen erhalten +30% FB. 
+ Man kann rev. Rebellen bekämpfen
+ Oder versuchen das reich lange genug zusammenzuhalten um die Revolution anzunehmen. 
![](https://forum.paradoxplaza.com/forum/index.php?attachments/upload_2019-12-3_9-34-4-png.529711/)
+ Schnelle Option: Revolutionäres Disaster entweder auf Seiten der Revolutionäre gegen Reaktionäre kämpfen => werde revolutionär mit Boni und kann Rev. Ziel werden, oder auf Seiten der Reaktionäre gegen Revolutionäre, verliert man wird man trotzdem Revolutionär aber mit Mali und null Eifer
+ Revolutionärer Eifer ersetzt Absolutismus: Max abhängig von Regierung, Großmacht, Golden Era, Revolution im Land. Erhöht durch Stärken Regierung, Rev Rebellen in anderen Ländern unterstützen, rev. Krieg
+ Aktuelle Boni (werden wahrscheinlich noch anders): +40% AdminEff, +100% Regierungsfortschritt, +50% Verstärkungsrate
+ Nur ein rev. Ziel, wenn Eifer zu gering ist kann andere Rev. Republik übernehmen, nur Ziel kann Empire/Diktatur werden
+ Wählt man im Disaster die Reaktionäre und gewinnt, ist das Land sicher vor Revolution. 
+ Kann auch Rev. Ziel angreifen und Revolution beenden, kann aber innerhalb von zwanzig Jahren wieder auferstehen, danach gilt Revolution als gescheitert.
+ 20 Eigene Regierungsreformen
